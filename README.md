# Priya

Priya is a locally-hosted, LLM-agnostic, project-agnostic agentic operating system.

You talk to one agent — **Priya**, the Orchestrator (CEO) — by text or voice. She delegates work
to a set of Managers (Engineering, Security, Social Media, User Research, Fundraising, …), who use
a target project's own agents, skills, commands, and workflows to get things done. Everything is
visible in a dark-themed dashboard centered on a 3D memory graph, and all long-term memory lives in
a plain-markdown Obsidian vault you can open, read, and edit yourself.

First target project: [AgentKavach](https://github.com/prrockzed/AgentKavach).

See [PLAN.md](PLAN.md) for the full design: architecture, org model, task lifecycle, runner
adapters, memory system, dashboard UI, and build phases.
